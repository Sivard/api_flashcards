# Add gem

If you want using this gem, you need add 

 `gem 'api_flashcards' , git: 'git://github.com/Sivard/api_flashcards.git'`

# Mounting

Mount engine to app

`mount ApiFlashcards::Engine, at: "/api_flashcards"`

to `config/routes.rb`
